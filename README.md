# DemoClarion

This project provides a demo on how using the [Clarion](https://sites.google.com/site/clarioncognitivearchitecture/) cognitive architecture for building an intelligent agent at the [WorldServer3D virtual environment](https://github.com/CST-Group/ws3d). It is a demonstration used at [IA941 - Laboratory of Cognitive Architectures](https://gudwin.fee.unicamp.br/novo/courses/IA941), a graduate-level course at the University of Campinas - Brazil.
